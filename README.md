# SendBulkMail
Send bulk mail (more than 10000 list name) using Power automate

### Paint Point

ต้องส่งเมลให้คนในองค์กร 15,000 ฉบับ มีการแบ่งแล้ว 500 ฉบับต่อการส่ง 1 ครั้ง แปลว่าต้องกดส่งเมล 30 ครั้ง เยอะเกินไป

### Need
ต้องการให้ใช้ RPA (Power Automate) มาส่งเมล โดยมี
- หน้า UI พิมพ์ข้อความ วางลิงก์ได้ ก่อนส่งเมล
- ส่งเมลแบบ auto
- ปลอดภัยกับข้อมูล

### Solution
ใช้วิธีที่ 2 ลูป 5000 ต่อ 4 วัน สามารถส่งรายบุคคลและเรียนคุณ……. ได้ด้วย แต่ใช้เวลาในการส่ง **เกือบ 1 ชม. (พี่แอนไม่ติดค่ะ)**

ต้องใช้
* SharePoint : เก็บข้อมูล(รายชื่อ, ที่อยู่อีเมล)
* Power App : หน้า UI สามารถกรอกข้อมูลเนื้อหาในนี้เลย และกดส่ง
* Power Automate : ใช้ในการส่งเมลอัตโนมัติ

### ข้อควรปฏิบัติสำคัญ (Important)

เมื่อระบบเริ่มทำงาน จะพบรายการอีเมลปรากฏในกล่อง **จดหมายร่าง (Drafts)** ใน Outlook 
-  ระบบจะทยอยส่งอีเมลออกจากกล่อง Drafts ให้เองโดยอัตโนมัติ เพื่อป้องกันไม่ให้ระบบอีเมลขัดข้องจากการส่งปริมาณมากพร้อมกัน
-  ระบบทำงานอัตโนมัติ เมื่อการส่งเสร็จสิ้น รายการในกล่อง Drafts จะลดลงและย้ายไปยังกล่อง **รายการที่ส่งแล้ว (Sent Items)** จนครบถ้วน
